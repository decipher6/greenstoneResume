# Greenstone Talent AI

Intelligent candidate screening platform with AI-powered resume analysis.

## Quick Links

- **Frontend**: https://greenstone-resume.vercel.app
- **Deployment Guide**: See `DEPLOYMENT_FINAL.md`
- **Render Quick Reference**: See `RENDER_QUICK_REFERENCE.md`

## Features

- ✅ Job post creation and management
- ✅ Bulk CV upload (up to 50 files: .pdf, .docx, .doc)
- ✅ AI-powered candidate scoring with Groq LLaMA 3.3
- ✅ Candidate profiles with detailed analytics
- ✅ Reports and dashboard
- ✅ Simplified authentication

## Tech Stack

- **Frontend**: React + Vite + Tailwind CSS
- **Backend**: FastAPI + Python 3.11
- **Database**: MongoDB Atlas
- **AI**: Groq API (LLaMA 3.3)
- **Hosting**: Vercel (Frontend) + Render (Backend)
